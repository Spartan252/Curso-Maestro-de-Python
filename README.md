# Python

Lenguaje interpretado de alto nivel, orientado a objetos y tipado dinámico.

## Características principales
- **Sintaxis simple y legible**: diseñada para ser fácil de leer y escribir.
- **Tipado dinámico**: no necesitas declarar el tipo de una variable.
- **Multiplataforma**: corre en Windows, macOS y Linux sin cambios.
- **Gran ecosistema**: miles de librerías disponibles vía `pip`.
- **Multiparadigma**: soporta programación orientada a objetos, funcional e imperativa.

## Conceptos clave
- **Variables y tipos**: `int`, `float`, `str`, `list`, `dict`, `tuple`, `set`
- **Control de flujo**: `if/elif/else`, `for`, `while`
- **Funciones**: `def` y funciones lambda
- **Clases y objetos**: herencia, encapsulamiento, polimorfismo
- **Manejo de errores**: `try/except/finally`
- **Módulos**: organización del código en archivos reutilizables

## Usos comunes
- Inteligencia Artificial y Machine Learning (`TensorFlow`, `PyTorch`)
- Desarrollo web (`Django`, `FastAPI`, `Flask`)
- Análisis de datos (`Pandas`, `NumPy`, `Matplotlib`)
- Automatización y scripting
- Ciencia e investigación
